Su

     A fake artificial intelligence who hates everything and everyone. The project use pyTelegramBotAPI. Su speak russian. She can send her foto to you, send meme to you, say hello, say goodbye, etc. If she don't understand you she say a random phrase.

     To run code you must install modules from requirements.txt by this command:

pip install -r requirements.txt

     GitHub repositorie: https://github.com/fisicpy/su-bot/

Simple game

    A simple game in which a cyan cube must collect magenta apples. The project use pygame. In file config.py is some setting for this project, but I advise you don't change this settings because the code may not run. In main.py there is main code of project. We use module keyboard because pygame.key doesn't allow us to hold buttons on keyboard.

     To run code you must install modules from requirements.txt by this command:

pip install -r requirements.txt

GitHub repositorie: https://github.com/fisicpy/simple-game

Arduino tennis

    Arduino tennis on I2C display 128x64 pixels. You must take all libraries from a folder "libries" to "C:\Program Files\Arduino\libraries" and connect all companents:

arduino A5 - display SCL
arduino A4 - display SDA
arduino 5V - display VCC
arduino GND - display GND
button right - arduino GND
button right - arduino D9
button left - arduino GND
button left - arduino D10

GitHub repositorie: https://github.com/fisicpy/arduino-tennis
